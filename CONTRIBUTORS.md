## Adding Yourself

To add yourself to the contributors list:

1. Make a contribution to the project
2. Comment on your PR or issue with:
   ```
   @all-contributors please add @username for code, docs
   ```
3. The bot will create a PR to add you to this file

### Contribution Types

Use these keywords when requesting to be added:

- `code` - Code contributions
- `doc` - Documentation
- `design` - Design and branding
- `ideas` - Ideas and planning
- `infra` - Infrastructure and DevOps
- `maintenance` - Repository maintenance
